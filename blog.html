What is the DOM in JavaScript?

The DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents the page structure as a tree of objects that JavaScript can use to read, modify, and manipulate elements, attributes, and content dynamically.

How do you select an element by its ID?

You can select an element by its ID using `document.getElementById('elementId')`. It returns the first element with the given ID, allowing you to manipulate it with JavaScript.

How do you change the text of an HTML element?

To change text content, use `element.innerText = "New Text"` or `element.textContent = "New Text"`. This updates the visible text inside the selected element.

What is `addEventListener` used for?

`addEventListener` attaches an event to an HTML element, like a click or keypress. It lets you define what should happen when the event occurs without overwriting existing handlers.

How do you create and insert a new element in the DOM?

Use `document.createElement('tag')` to create an element, then use `parent.appendChild(newElement)` or `parent.insertBefore()` to insert it into the DOM.
